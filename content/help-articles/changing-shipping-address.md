# Changing your shipping address

You can update your saved address anytime, but changing an address on an order in progress depends on how far along the order is.

## Update your saved addresses

1. Sign in and go to **Account → Addresses**.
2. Click **Add address** or **Edit** next to an existing one.
3. Save your changes. The default address is used for future orders.

## Change the address on an existing order

| Order status | Can you change the address? |
|--------------|------------------------------|
| Pending / Processing | Yes — edit directly from the order page. |
| Awaiting fulfillment | Yes — contact support immediately. |
| Shipped | No — the package is with the carrier. |
| Delivered | No |

To edit an in-progress order:

1. Open **Orders** and select the order.
2. If **Change address** is visible, click it and update.
3. If the button isn't visible, the order has already entered fulfillment. Contact support — we can sometimes intercept it.

## Wrong address on a shipped package

If a package is already in transit:

- **Same city / region:** most carriers offer a paid address-change service. Use the carrier's tracking page to request a redirect.
- **Different country:** the package will either be returned to sender or held at the destination. Wait for it to return, then we'll re-ship to the correct address (you cover the new shipping fee).

## PO boxes and military addresses

PO boxes work for standard shipping but not express. APO/FPO/DPO addresses are supported — select **United States** as the country and use the appropriate state code (AA, AE, AP).
