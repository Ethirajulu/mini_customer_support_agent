# Applying a promo code

Promo codes give you a discount, free shipping, or a free gift at checkout.

## How to apply a code

1. Add items to your cart and go to **Checkout**.
2. Find the **Promo code** field on the order summary.
3. Type the code exactly as written (codes are case-sensitive) and click **Apply**.
4. The discount appears on the order total before you pay.

## Code not working?

Check these common reasons:

- **Typo or whitespace.** Codes are case-sensitive and shouldn't have leading or trailing spaces.
- **Expired.** Codes have a validity window. The expiration date is usually in the email or promo you received it from.
- **Minimum order not met.** Many codes require a minimum subtotal.
- **Excluded items.** Sale items, gift cards, and certain brands may be excluded.
- **Already used.** Single-use codes only work once per account.
- **Region restriction.** Some promos are limited to specific countries.

## Stacking codes

Only one promo code per order. To combine offers, look for automatic discounts (e.g., free shipping over $50) — these apply on top of a single code.

## Can I apply a code after I ordered?

We can apply a code retroactively if:

- The order was placed within the last **48 hours**.
- The order hasn't shipped.
- The code was valid at the time of purchase.

Contact support with your order number and the code to request the adjustment.

## First-order discount

The 10% off welcome code is sent to your email after you sign up. If you never received it, check your spam folder or contact us — we can resend.
